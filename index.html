Python / Natural Language Processing / Machine Learning

Resume Parser

Finding qualified applicants for an open position can be challenging, particularly when a large number of applicants from various backgrounds apply for the position. This project focuses on finding the similarity between the job description and resume and it outputs the best suitable job title along with the probability for each of the resumes. This method also helps recruiters when the candidate applies for multiple roles within the company by simply looking at the percentages, they can determine which role the candidate’s resume best fits.We fitted the random forest classifier to the resume dataset to predict which category the resume belongs to and compared it to the actual category column.Additionally, we created a working system, where a job description and resume are uploaded,the system calculates the probability to which they match and finds top three categories that best suits the resume.
